# Meme-Generator

Esse projeto me permitiu ultilizar todos os conhecimentos básicos de desenvolvimento web com ferramentas de frontend, utilizei JavaScript, CSS, HTML e até mesmo o DOM.
Consiste em um site que gera memes, citações, piadas e ate mesmo charadas com respostas, a interface do projeto foi feita com flexbox e permite a vizualização dos elementos
HTML que são gerados a partir de botões que executam uma função em JavaScript que se comunica com a estrutura de HTML a partir do DOM.
